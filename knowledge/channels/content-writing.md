# Content Writing Guide

> **Use when:** Writing blog posts, articles, or any long-form content for marketing purposes.

## Quick Reference

- Blogs: conversational, first-person, 1,500-2,500 words for SEO
- Articles: formal, third-person, 600-1,500 words, evidence-based
- Hook readers in first 2-3 sentences with statistics, questions, or bold statements
- Use headers every 200-300 words, 2-4 sentences per paragraph
- Apply the Four U's: Unique, Useful, Ultra-specific, Urgent

---

## Blog vs. Article: When to Use Each

| Factor | Blog Post | Article |
|--------|-----------|---------|
| **Tone** | Conversational, casual | Formal, professional |
| **Voice** | First person ("I", "we") | Third person, objective |
| **Purpose** | Attract customers, build brand | Deliver structured knowledge |
| **Audience** | General audiences, early-stage buyers | Experts, professionals, executives |
| **Length** | 1,000-2,500 words | 600-1,500 words |
| **Research** | Personal insights + supporting data | Heavy research, citations |

**Choose BLOG when:**
- Building brand personality
- Sharing opinions, experiences, or behind-the-scenes content
- Driving SEO traffic
- Creating shareable content

**Choose ARTICLE when:**
- Establishing thought leadership
- Presenting research or data analysis
- Writing for publications or professional audiences
- Creating evergreen reference material

## Introduction Hooks (7 Types)

### 1. Surprising Statistic
```
Only 2% of website visitors convert on their first visit. The other 98%?
They're gone — unless you have a content strategy that brings them back.
```

### 2. Provocative Question
```
What if everything you've been told about SEO is wrong?
```

### 3. Bold Statement
```
Email marketing isn't dying. Your email marketing is dying.
```

### 4. Personal Anecdote
```
Last Tuesday, I accidentally deleted our entire content database.
What happened next taught me more about backup systems than any
tutorial ever could.
```

### 5. Pain Point
```
You've spent hours crafting the perfect blog post. You hit publish.
And then... nothing. No traffic. No shares. Sound familiar?
```

### 6. Contrarian Take
```
Most content marketing advice tells you to post more. I'm going to
tell you to post less — and here's why that's the key to growth.
```

### 7. Promise of Value
```
In the next 10 minutes, you'll learn the exact framework that helped
us grow organic traffic by 400% in 6 months.
```

## Structure Templates

### Problem-Solution (How-To/Guides)
1. Establish the problem/pain point
2. Present the solution framework
3. Step-by-step implementation
4. Results/proof

### Inverted Pyramid (News/Articles)
1. Most important information first
2. Supporting details second
3. Background/context last

### Listicle Structure
1. Brief introduction with promise
2. Numbered/bulleted main points
3. Conclusion with next steps

## Formatting Rules

### Paragraphs
- **2-4 sentences max** per paragraph
- **One idea per paragraph**
- Use line breaks generously
- Short paragraphs for mobile

### Headers
- Use every 200-300 words
- Make headers descriptive (frontload keywords)
- Headers should work as standalone outline
- H1 → H2 → H3 hierarchy

**Good Headers:**
```
## How to Write Headlines That Convert
### Step 1: Research Your Audience's Pain Points
```

**Bad Headers:**
```
## Introduction
## Main Points
## Conclusion
```

### Bullet Points
- Start each with same part of speech
- Keep approximately equal length
- Use parallel structure
- Limit to 5-7 before breaking into subsections

## Writing Style

### Voice
- **Active voice** (90%+): "Our team analyzed 1,000 posts"
- **Avoid passive**: "1,000 posts were analyzed by our team"

### Readability Targets
| Metric | Target |
|--------|--------|
| Reading Level | 6th-8th grade |
| Sentence Length | 15-20 words average |
| Paragraph Length | 2-4 sentences |
| Passive Voice | Under 10% |

### Words to Avoid
| Avoid | Use Instead |
|-------|-------------|
| Utilize | Use |
| Leverage | Use, Apply |
| Synergy | Collaboration |
| Optimize | Improve |
| Innovative | New |
| Basically | (Delete) |
| Actually | (Delete) |
| Very/Really | (Find stronger word) |

## SEO Integration

### Keyword Placement
- Title/H1 (near beginning)
- First paragraph (within first 100 words)
- At least one H2 subheading
- Throughout body (1-2% density)
- Meta description
- URL slug
- Image alt text

### Featured Snippet Optimization
```
## What Is Content Marketing?

Content marketing is a strategic approach focused on creating and
distributing valuable, relevant content to attract and retain a
clearly defined audience — ultimately driving profitable customer action.

Key characteristics include:
- Audience-focused (not product-focused)
- Consistent publishing schedule
- Value-driven (educates, entertains, or solves problems)
```

### Internal Linking
- Link within first 100-200 words
- Use descriptive anchor text (not "click here")
- Link to pillar content frequently
- Aim for 3-5 internal links per 1,000 words

## Engagement Techniques

### Reader Focus
```
Good: "You're about to learn the framework that changed everything."
Bad: "We're excited to announce our new framework."
```

### Storytelling Structure
1. **The Problem** — Reader faces a challenge
2. **The Struggle** — Previous attempts haven't worked
3. **The Discovery** — A new approach emerges
4. **The Transformation** — Implementation and change
5. **The Result** — Success and lessons learned

### Curiosity Gaps
```
"The third technique surprised even our most experienced marketers..."
"But here's where most people get it wrong..."
"What happened next changed our entire strategy..."
```

## Length Guidelines

| Content Type | Word Count | Purpose |
|--------------|------------|---------|
| News/Updates | 300-600 | Quick announcements |
| Standard Blog | 1,000-1,500 | Topic overviews |
| SEO-Optimized | 1,500-2,500 | Comprehensive coverage |
| Ultimate Guides | 2,500-4,000+ | Pillar content |
| Professional Articles | 600-1,200 | Trade publications |

**The right length is:**
- Long enough to thoroughly answer the question
- Short enough to maintain engagement
- Dense with value (no filler)

## Checklist

### Pre-Writing
- [ ] Clear target audience defined
- [ ] Specific problem/question identified
- [ ] Keyword research completed
- [ ] Competitive content reviewed
- [ ] Unique angle determined

### During Writing
- [ ] Hook in first 2-3 sentences
- [ ] One main idea per paragraph
- [ ] Active voice used (90%+)
- [ ] Headers every 200-300 words
- [ ] Examples and specifics included

### Post-Writing
- [ ] Read aloud for flow
- [ ] Readability score checked (8th grade or below)
- [ ] Spelling and grammar verified
- [ ] All claims fact-checked
- [ ] Links tested
- [ ] Images optimized with alt text
- [ ] Meta description written
- [ ] CTA included

### Final Review
- [ ] Would I share this with a colleague?
- [ ] Does this offer something competitors don't?
- [ ] Could a beginner understand this?
- [ ] Can I grasp main points in 30 seconds (scanability)?
- [ ] Does the reader know what to do next?
