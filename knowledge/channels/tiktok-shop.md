# TikTok Shop Content Guide

> **Use when:** Creating shoppable content for TikTok Shop, including product videos, affiliate content, or creator commerce campaigns.

## Quick Reference

- TikTok is a **visual search engine**, not a social media app - optimize for keywords
- Use the 6-Step Script: Hook → Soft CTA → Problem → Demo → FOMO → Hard CTA
- Say AND show the keyword in the first 3 seconds
- Place a "Soft CTA" (mention the shop/price) within 15 seconds
- Use specific hashtags (#coffeelover), NOT generic ones (#fyp, #viral)

---

## Core Principle: TikTok is a Search Engine

TikTok operates on two algorithms:
1. **Entertainment Algorithm:** Watch time, shares, emotional engagement
2. **Shopping Algorithm:** Search intent, CTR, conversion density

**For commerce, optimize for the Shopping Algorithm.** The platform:
- Transcribes audio (speech-to-text)
- Reads text overlays (OCR)
- Scans metadata and hashtags

Your goal is **indexable visibility** - ensuring content is categorized correctly to reach users with purchase intent.

## Product Selection Criteria

### The Three-Second Mute Test
Can the primary benefit be understood in under 3 seconds with sound off?

| Product Type | Pass/Fail | Reasoning |
|--------------|-----------|-----------|
| Hover Skates | **Pass** | Motion alone stops scroll |
| Generic Supplement | **Fail** | Bottle of pills is static, needs explanation |
| Stain Remover | **Pass** | Before/After transformation is visual |
| Standard Coffee Maker | **Fail** | No visual differentiation |
| Unique Brewing Device | **Pass** | Unique mechanism holds attention |

### Winner Criteria
- **Visual Appeal / Wow Factor:** Must arrest attention immediately
- **Solves Real Problem:** Pre-existing frustration, not manufactured need
- **Brand Story:** Differentiation beyond price
- **Margin:** High-ticket or high-commission to maximize revenue per view

## The 6-Step Scripting Framework

### Step 1: Search-Indexed Hook (0:00-0:03)

**Objective:** Stop scroll AND index for SEO

Synchronize three layers:
- **Verbal:** Say the keyword in first 3 seconds
- **Textual:** Keyword as text overlay
- **Visual:** Product visible in frame

**Example:**
```
"If you love coffee but hate spending $5 a cup, stop scrolling right now."
```

### Step 2: Soft Call to Action (0:03-0:15)

**Objective:** Seed the purchase location early, bypass "ad detector"

**Key Phrases:**
```
"I found this on TikTok Shop the other day..."
"I can't believe it was this cheap..."
"When I grabbed it, it was under $20..."
```

**Why it works:** Frames you as a "finder" of deals, not a "seller" of products. Tells viewer WHERE to buy without being aggressive.

### Step 3: Problem Agitation (0:15-0:30)

**Objective:** Validate product's existence through pain point

**Example:**
```
"I've been dealing with [Problem X] for years, and I saw everyone
talking about this solution, so I finally caved and bought it."
```

**Why it works:** Shifts narrative from "I am selling you" to "I am joining the community that uses this."

### Step 4: Wow Demonstration (0:30-0:45)

**Objective:** Visual proof of efficacy

- Script effectively stops - visual takes over
- No talking head - just product in action
- Use ASMR sounds or clear voiceover

**Example:**
```
"Look at how it instantly [Action]. It took less than ten seconds."
```

### Step 5: FOMO/Scarcity Anchor (0:45-0:50)

**Objective:** Create urgency for immediate click

**Examples:**
```
"I don't know if the flash sale is still running..."
"It says 'Low Stock' right now..."
"When I grabbed it, it was under $20 - not sure if that's still the price"
```

**Why it works:** Creates curiosity gap. Viewer clicks to check price even if not fully committed.

### Step 6: Hard Call to Action (0:50-0:60)

**Objective:** Direct instruction to interface

**Example:**
```
"Click the orange cart right there [point to screen] before they sell out again."
```

## Filming & Editing: "Clean Cut" Method

### Retention Engineering Pacing

**Zero-Breath Rule:** No dead air. Cut every intake of breath, every "um."

**Visual Reset Rate:** Change angle, zoom, or scene every 2-4 seconds.

### Modular Hook Stacking

Film in blocks for A/B testing:
- **Hook Block (0-5s):** Film 3-4 different hooks
- **Body Block (5-50s):** Core demonstration
- **CTA Block (50-60s):** Closing

If video fails, swap Hook Block and repost.

### Required Shot List
1. **A-Roll:** Creator holding product (The Hook)
2. **Macro B-Roll:** Extreme close-up of texture/details
3. **Action Shot:** Product in use (the "Wow")
4. **Reaction Shot:** Creator's face showing satisfaction

### "Good Quality" Technical Requirements

Videos tagged "Good Quality" get 50-70% view boost:

- [ ] **Lighting:** Subject and product well-lit, no shadows/grain
- [ ] **Audio:** Clear and crisp (use lavalier mic, not phone mic)
- [ ] **Safe Zones:** Text never obscures Like/Comment/Share buttons

## Posting & SEO Strategy

### Keyword Loading

The algorithm transcribes audio, so the script IS the SEO strategy.

**Caption Strategy:**
```
Bad: "Love this! 😍"
Good: "This portable espresso maker is the best travel coffee hack for camping."
```

### Hashtag Logic

**Avoid:** #fyp, #foryou, #viral (too broad, ruins conversion)

**Use:** Specific hashtags that act as database queries
```
#coffeelover #espressohack #kitchengadgets #momlife #dadtech
```

### The "Live" Catalyst

Going live signals active participation:
- Go live for 60-90 minutes
- Creates "halo effect" - drives traffic to static videos
- Use lives to warm up algorithm before posting new content

### Posting Frequency

Quality over quantity:
- Better: 1 "Good Quality" video per day
- Worse: 5 low-quality videos that hurt account score

## Audit Checklist

| Component | Audit Question | Pass/Fail |
|-----------|----------------|-----------|
| **Hook (0-3s)** | Said AND showed keyword? | Fail if keyword missing from text or audio |
| **Pacing** | Any silence > 0.5s? | Fail if dead air exists |
| **Soft CTA** | Mentioned shop/price within 15s? | Fail if CTA only at end |
| **Visuals** | Product demonstrated in action? | Fail if just talking head |
| **Lighting** | Product clearly visible and sharp? | Fail if grainy or dark |
| **SEO** | Hashtags specific to niche? | Fail if #fyp or #viral used |
| **Compliance** | Medical/weight loss claims? | Fail = immediate violation risk |

## Hook Formulas

### The "Discovery" Hook
```
"I found this on TikTok Shop and I can't believe..."
"I didn't think this $20 [product] would actually work..."
```

### The "Negative Assumption" Hook
```
"I was skeptical, but..."
"I didn't expect this to work, but..."
```

### The "If You" Hook
```
"If you love [activity] but hate [pain point], stop scrolling."
"If you've tried everything for [problem] and nothing works..."
```

### The "Everyone's Talking" Hook
```
"I kept seeing this everywhere so I finally tried it..."
"Everyone's obsessed with this and I get why now..."
```
