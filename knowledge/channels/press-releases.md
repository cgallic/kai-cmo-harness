# Press Release Writing Guide

> **Use when:** Drafting announcements for product launches, funding rounds, partnerships, or milestones for distribution to media outlets.

## Quick Reference

- Follow exact structure: FOR IMMEDIATE RELEASE → Headline → Lead → Body → Quote → Boilerplate
- Lead paragraph: 5 Ws + "So What" in 25-40 words
- Ideal length: 400-500 words
- Use AP Style for all formatting
- Include one executive quote, optionally one customer/partner quote

---

## Press Release Structure

```
FOR IMMEDIATE RELEASE

[HEADLINE - Title Case, Under 70-100 Characters]

[Optional Subheadline - Key detail]

[CITY], [State Abbr.] (Month Day, Year) — [Lead paragraph: 5 Ws in 25-40 words]

[Body paragraph 1 - Supporting details]

[Body paragraph 2 - Additional context]

"[Quote from executive]," said [Name], [Title] of [Company].

[Body paragraph 3 - Features/benefits, can use bullets]

[Optional second quote from customer/partner]

[Call to action paragraph]

###

About [Company]
[Boilerplate - 3-4 sentences]

Media Contact:
[Name]
[Email]
[Phone]
```

## AP Style Rules (Mandatory)

### Numbers
- Spell out: one through nine
- Use numerals: 10, 11, 100, 1,000
- Always numerals for: ages, percentages, money, dates
- Examples: "five users" but "10 users" | "$5 million" | "25%"

### Dates
- Abbreviate: Jan., Feb., Aug., Sept., Oct., Nov., Dec.
- Spell out: March, April, May, June, July
- Format: Dec. 19, 2024 (no "th" or "st")

### Punctuation
- **NO Oxford comma**: "red, white and blue" NOT "red, white, and blue"
- Single space after periods
- Periods and commas go inside quotation marks

### Titles
- BEFORE name (capitalize): Chief Executive Officer John Smith
- AFTER name (lowercase): John Smith, chief executive officer of Company
- Common abbreviations: CEO, CFO, CTO (no periods)

### Acronyms
- Define on first use: "artificial intelligence (AI)"
- Well-known acronyms don't need definition: FBI, NASA, CEO

### Cities That Stand Alone (No State Needed)
Atlanta, Boston, Chicago, Dallas, Denver, Houston, Las Vegas, Los Angeles, Miami, New York, Philadelphia, San Francisco, Seattle, Washington

## Headline Formulas

### Launch Announcements
```
[Company] Launches [Product] to [Solve Problem/Enable Outcome]
[Company] Unveils [Product], [Key Differentiator]
[Company] Introduces [Product] for [Target Audience]
```

### Funding Announcements
```
[Company] Raises $[X]M to [Mission/Expansion Goal]
[Company] Closes $[X]M [Round Type] Led by [Lead Investor]
[Company] Secures $[X]M in [Round Type] to [Accelerate/Build]
```

### Partnership Announcements
```
[Company] Partners with [Partner] to [Outcome/Integration]
[Company] and [Partner] Announce [Type of Partnership]
[Company] Integrates [Partner Technology] to [Benefit]
```

### Milestone Announcements
```
[Company] Surpasses [Metric] [Users/Volume]
[Company] Reaches [Milestone], [Key Achievement Detail]
```

## Lead Paragraph Template

Must answer 5 Ws + "So What" in 25-40 words:

```
[CITY], [State] ([Date]) — [Company name], [brief descriptor], today
announced [what], [enabling/providing/delivering] [benefit to users/market].
```

**Example:**
```
NEW YORK, Dec. 19, 2024 — Indexify, a social terminal for crypto
portfolios, today launched its beta platform on Solana, enabling
traders to create transparent, on-chain portfolios and earn fees
when others follow their strategies.
```

**Checklist:**
- [ ] WHO: Company name included
- [ ] WHAT: Announcement clearly stated
- [ ] WHEN: Date in dateline
- [ ] WHERE: City in dateline
- [ ] WHY: Benefit or purpose stated
- [ ] SO WHAT: Why readers should care

## Quote Best Practices

### Executive Quote (Required)

**Template:**
```
"[Insight about problem or market] [How product solves it] [Vision
for future]," said [Full Name], [title] of [Company].
```

**Good Example:**
```
"Crypto traders have been stuck managing chaos across ten narratives
and twenty tokens with no clear way to measure what's working," said
Sebastian Frankel, co-founder of Indexify. "We built Indexify to turn
that noise into signal that others can follow and verify."
```

**Bad Example:**
```
"We are excited to launch this revolutionary product that will
disrupt the industry," said CEO John Smith.
```

### Quote Rules
- Never start quotes with "I" if avoidable
- Avoid buzzwords: synergy, leverage, optimize, revolutionary
- Include specific details or metrics
- Quotes should add insight, NOT repeat body text
- Keep to 2-3 sentences maximum

## Crypto/Fintech Guidelines

### Do's
- Lead with verifiable metrics (TPS, TVL, user count)
- Name integration partners explicitly
- Emphasize self-custody and transparency
- Include on-chain verification details
- Reference audits or security measures
- Mention regulatory compliance where relevant

### Don'ts
- Avoid hype words: "Revolutionary," "Groundbreaking," "Game-changing"
- Don't use "First-ever" unless verifiably true
- No price predictions or investment advice
- Avoid unverifiable superlatives
- Don't compare to competitors negatively

### Terminology
- Use "self-custody" not "non-custodial"
- Use "on-chain" (hyphenated as adjective)
- Define DeFi, DEX, TVL on first use for mainstream outlets
- Specify blockchain: "on Solana" not "on blockchain"

### Trust Signals to Include
- Audit status and auditor name
- Integration partners
- Investor names (if disclosed)
- Team credentials (if notable)
- User/volume metrics

## Boilerplate Template

3-4 sentences covering:
1. What the company does
2. Key differentiator
3. Traction indicator (optional)
4. Website

```
About [Company]
[Company] is [what it does]. The [platform/company] [key differentiator
or how it works]. [Optional: traction metric]. For more information,
visit [website].
```

**Example:**
```
About Indexify
Indexify is the social terminal for crypto portfolios on Solana. The
platform transforms market ideas into transparent, on-chain portfolios
called Stacks that anyone can create, follow, or trade. Combining social
discovery with self-custodied execution and AI-powered intelligence,
Indexify is building the next layer of creator-powered trading
infrastructure. For more information, visit indexify.finance.
```

## Pre-Distribution Checklist

### Content
- [ ] Headline under 100 characters with action verb
- [ ] Lead answers 5 Ws in under 40 words
- [ ] All facts and figures verified
- [ ] Quotes attributed with full name and title
- [ ] AP style applied throughout
- [ ] No spelling or grammar errors
- [ ] Links tested

### Format
- [ ] "FOR IMMEDIATE RELEASE" header
- [ ] Dateline correct: CITY, State (Month Day, Year) —
- [ ] Media contact complete
- [ ] Boilerplate included
- [ ] ### end mark present

### Multimedia
- [ ] High-res images available (300 DPI min)
- [ ] Product screenshots prepared
- [ ] Executive headshots available

### Distribution
- [ ] Target publication list prepared
- [ ] Journalist contacts identified
- [ ] Social snippets drafted
- [ ] Timing confirmed (Tue-Thu, 9-11am optimal)
