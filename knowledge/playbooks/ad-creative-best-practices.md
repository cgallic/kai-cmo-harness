# Ad Creative Best Practices — Cross-Platform Playbook

> **Use when:** Creating ad creatives for any platform. Covers copy, visuals, video, and format-specific optimization.

---

## Universal Creative Principles

### The 3-Second Rule
Users decide in 3 seconds whether to engage. Every ad must pass this test:
1. **Hook** — First line/frame grabs attention
2. **Value** — Clear benefit visible immediately
3. **Action** — Obvious next step

### Creative Hierarchy
```
STRONGEST → WEAKEST (by conversion impact)
1. Offer (what you're selling and at what price)
2. Creative (the visual/video asset)
3. Headline (the text that frames the creative)
4. Body copy (supporting text)
5. CTA button (the action prompt)
```

An irresistible offer with mediocre creative will outperform brilliant creative with a weak offer every time.

### Copy Formulas That Convert

**PAS — Pain, Agitate, Solution:**
- Pain: Name the specific problem
- Agitate: Show what happens if they don't fix it
- Solution: Present your product as the fix

**AIDA — Attention, Interest, Desire, Action:**
- Attention: Pattern interrupt (stat, question, bold claim)
- Interest: Expand on the hook with specifics
- Desire: Social proof or transformation story
- Action: Clear, single CTA

**BAB — Before, After, Bridge:**
- Before: Current painful state
- After: Desired outcome achieved
- Bridge: Your product is how they get there

## Platform-Specific Creative Rules

### Meta (Facebook/Instagram)

**Image Ads:**
- 1:1 ratio for feed, 9:16 for Stories/Reels
- Less than 20% text on image (soft rule, still impacts delivery)
- High contrast — must pop on a white/dark feed background
- Human faces outperform product shots by 30-50%
- UGC-style (phone shot, natural lighting) outperforms polished studio

**Video Ads:**
- First 3 seconds = hook or bust (autoplay, sound off)
- Design for sound-off viewing — use captions/text overlays
- Optimal length: 15-30 seconds for awareness, 30-60 for consideration
- Vertical (9:16) for Reels/Stories, square (1:1) for feed
- End with clear CTA card (last 3 seconds)

**Copy Structure:**
- Primary text: 125 chars recommended (can go to 2200 but truncates)
- Headline: 40 chars max
- Description: 30 chars max
- Use emojis sparingly — 1-2 max, at line breaks for visual scanning

**A/B Test Priority (test one at a time):**
1. Hook/opening line (highest impact)
2. Creative asset type (image vs video vs carousel)
3. Offer framing (discount vs value vs urgency)
4. Audience (last — let the algorithm find your audience)

### Google Ads

**Search Ads:**
- Headlines: 30 chars each, 15 headline slots available
- Descriptions: 90 chars each, 4 description slots
- Include keyword in headline 1 (Quality Score impact)
- Use numbers: "Save 37%" > "Save Money"
- Power words in headlines: Free, New, Proven, Guaranteed, Instant
- Pin best-performing headlines to position 1

**Display Ads:**
- Responsive display ads outperform uploaded image ads
- Provide 5+ headlines, 5+ descriptions, 5+ images, 2+ logos
- Mix short headlines (25 chars) with long headlines (90 chars)
- Images: 1200x628 landscape, 300x300 square minimum

**Performance Max:**
- Provide all asset types (text, image, video)
- Write for each funnel stage (awareness, consideration, conversion)
- Use audience signals, not audience targeting (it's a hint, not a constraint)
- Budget >= $50/day for meaningful data

### TikTok Ads

**Video Creative:**
- First 2 seconds: hook or skip (even faster than Meta)
- Native TikTok style outperforms polished ads by 3-5x
- Use trending sounds/music (check commercial license)
- Text overlays are mandatory — 85% watch sound-off in feed
- Optimal length: 9-15 seconds for TopView, 21-34 for In-Feed
- End screen with CTA + product

**Creative Formats:**
- Spark Ads (boost organic posts) — highest authenticity, lowest CPM
- In-Feed Ads — blend with organic content
- TopView — first ad user sees on open
- Branded Effects — interactive, high engagement

**Copy:**
- Ad text: 100 chars recommended
- Casual, conversational tone (not corporate)
- AI content disclosure required by platform TOS

### LinkedIn Ads

**Sponsored Content:**
- 150 chars intro text (truncates on mobile)
- Headline: 70 chars
- Description: 100 chars
- Image: 1200x627 (1.91:1 ratio)
- Professional context required — casual TikTok style won't work here

**Thought Leadership:**
- Data-driven claims perform best (cite specific stats)
- "Industry insight" framing > product pitch
- Name-drop recognizable companies/tools for credibility
- Question-led posts get 50%+ higher engagement than statements

## Creative Testing Framework

### The 3x3 Matrix

Test systematically. Run 3 hooks × 3 creatives = 9 variations per campaign.

```
           CREATIVE A      CREATIVE B      CREATIVE C
           (UGC video)     (Product shot)  (Testimonial)
┌─────────┬───────────────┬───────────────┬───────────────┐
│ HOOK 1  │  Variant 1    │  Variant 2    │  Variant 3    │
│ (Pain)  │               │               │               │
├─────────┼───────────────┼───────────────┼───────────────┤
│ HOOK 2  │  Variant 4    │  Variant 5    │  Variant 6    │
│ (Social)│               │               │               │
├─────────┼───────────────┼───────────────┼───────────────┤
│ HOOK 3  │  Variant 7    │  Variant 8    │  Variant 9    │
│ (Data)  │               │               │               │
└─────────┴───────────────┴───────────────┴───────────────┘
```

### When to Kill a Creative
- After 2x the expected CPA with no conversions → kill
- CTR < 0.5% after 1000 impressions → kill (for Meta)
- CTR < 1% for search ads → kill (for Google)
- Frequency > 3 on same audience → refresh creative

### Creative Fatigue Signals
- CTR dropping week-over-week (>15% decline)
- CPA rising week-over-week (>20% increase)
- Frequency climbing above 2.5
- Comment sentiment turning negative ("I keep seeing this")

## Ad Copy Checklist

Before any ad goes live:

- [ ] Hook passes the 3-second test (read first line aloud — would you stop scrolling?)
- [ ] Single clear CTA (not two competing actions)
- [ ] Offer is specific (not "learn more about our solution")
- [ ] Platform character limits respected (no truncation)
- [ ] No banned words from quality gate (leverage, synergy, etc.)
- [ ] Platform TOS compliance checked (load the relevant policy reference)
- [ ] Landing page matches ad promise (no bait-and-switch)
- [ ] Tracking UTMs set up (utm_source, utm_medium, utm_campaign, utm_content)
- [ ] Mobile preview checked (most impressions are mobile)
- [ ] A/B variant created (never run a single version)
- [ ] Budget set appropriately (minimum viable for learning phase)

## Budget & Bidding Best Practices

### Starting Budgets (per platform, per day)
| Platform | Minimum Viable | Recommended | Learning Phase |
|----------|---------------|-------------|----------------|
| Meta | $20/day | $50-100/day | 50 conversions/week |
| Google Search | $30/day | $75-150/day | 30 conversions/month |
| Google PMax | $50/day | $100-200/day | Needs volume |
| TikTok | $20/day | $50-100/day | 50 conversions/week |
| LinkedIn | $50/day | $100-200/day | Expensive — B2B only |

### Bidding Strategy Ladder
```
Phase 1 (Learning):  Manual CPC or Maximize Clicks
Phase 2 (Data):      Target CPA or Target ROAS (need 30+ conversions)
Phase 3 (Scale):     Maximize Conversions with CPA cap
Phase 4 (Mature):    Value-Based Bidding (need conversion value data)
```

Never skip phases. Algorithms need data before they can optimize.
