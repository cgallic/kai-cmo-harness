# Content Writing Checklist

> **Use when:** Final review before publishing any blog post or article.

## Pre-Writing
- [ ] Target audience clearly defined
- [ ] Specific problem/question identified
- [ ] Primary keyword researched
- [ ] Competitive content reviewed
- [ ] Unique angle determined
- [ ] Four U's scored (target 12+/16)

## Structure
- [ ] Hook in first 2-3 sentences
- [ ] Headers every 200-300 words
- [ ] One main idea per paragraph
- [ ] Paragraphs 2-4 sentences max
- [ ] Scannable subheads (work as outline)

## Writing Quality
- [ ] Active voice (90%+)
- [ ] Reading level 6th-8th grade
- [ ] Sentences average 15-20 words
- [ ] No jargon/buzzwords ("utilize," "leverage")
- [ ] Specific examples included
- [ ] Numbers used (not "many" or "several")

## SEO
- [ ] Keyword in first 100 words
- [ ] Keyword in at least one H2
- [ ] Meta description written (155 chars)
- [ ] URL slug includes keyword
- [ ] Internal links added (3-5 per 1,000 words)
- [ ] Image alt text descriptive

## Engagement
- [ ] Value promised in intro
- [ ] Reader-focused ("you") not author-focused ("we")
- [ ] Curiosity gaps created
- [ ] CTA included
- [ ] Next steps clear

## Final Review
- [ ] Read aloud for flow
- [ ] Spelling/grammar verified
- [ ] All claims fact-checked
- [ ] Links tested
- [ ] Would I share this with a colleague?
