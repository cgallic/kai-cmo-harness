# Creative Production Checklist

> **Use when:** Producing ad creatives, social media content, or any visual/video marketing assets.

---

## Image Creative Checklist

### Concept
- [ ] Clear single message (can you describe the ad's point in 5 words?)
- [ ] Target persona identified (which persona is this for?)
- [ ] Emotional trigger defined (fear of loss? aspiration? curiosity?)
- [ ] Differentiation from competitors (does this look/sound like no one else?)

### Copy
- [ ] Headline passes the "scroll test" (would you stop scrolling?)
- [ ] Character limits respected per platform
- [ ] No banned words (run through quality gate)
- [ ] CTA is single and specific (not "Learn more about our comprehensive solution")
- [ ] Proof point included (number, testimonial, or credential)

### Visual
- [ ] Correct dimensions for target platform and placement
- [ ] High contrast — readable on both light and dark backgrounds
- [ ] Text < 20% of image area (Meta soft rule, still impacts delivery)
- [ ] Faces/people included (human faces improve CTR 30-50%)
- [ ] Brand colors present but not overwhelming
- [ ] No stock photo clichés (handshake, pointing at graph, generic smile)
- [ ] Mobile preview: is everything readable at phone size?

## Video Creative Checklist

### Pre-Production
- [ ] Script written with timestamped sections
- [ ] Hook designed for first 1-3 seconds
- [ ] Shot list / storyboard created
- [ ] Equipment ready (camera, mic, lighting)
- [ ] Location scouted (background, noise level, lighting)
- [ ] Talent briefed on key messages

### Production
- [ ] Audio clear (lavalier mic or close-proximity mic)
- [ ] Lighting even (no harsh shadows on face)
- [ ] Background clean (no distracting elements)
- [ ] Multiple takes of each section (at least 3)
- [ ] B-roll captured (product shots, screen recordings, environment)
- [ ] Vertical (9:16) and horizontal (16:9) versions if needed

### Post-Production
- [ ] First 1-3 seconds hook confirmed (strongest version selected)
- [ ] Captions/subtitles added (85% watch without sound)
- [ ] Background music added (-15 to -20dB behind voice)
- [ ] Text overlays at key moments (reinforce spoken points)
- [ ] Pacing: cuts every 2-3 seconds for social, 5-8 for YouTube
- [ ] CTA end card (last 3-5 seconds)
- [ ] Color grading consistent across clips
- [ ] Audio levels normalized

### Platform-Specific
- [ ] TikTok: AI content disclosure added (if applicable)
- [ ] YouTube: Custom thumbnail created (not auto-generated)
- [ ] YouTube: Chapters/timestamps added to description
- [ ] Instagram: Cover frame selected (for Reels grid display)
- [ ] All: Music licensing verified for commercial use

## Carousel / Document Checklist

### Structure
- [ ] Slide 1: Hook (pattern interrupt — not a title card)
- [ ] Slides 2-9: One point per slide (clear, scannable)
- [ ] Last slide: CTA (save, share, follow, visit link)
- [ ] Total slides: 5-10 (sweet spot for engagement)

### Design
- [ ] Consistent visual style across all slides
- [ ] Large, readable text (min 24pt equivalent on mobile)
- [ ] One key takeaway per slide (not a wall of text)
- [ ] Brand colors/logo subtle but present
- [ ] Arrow or visual cue to "swipe" on first slide

---

## Quality Gate (All Creatives)

Before any creative ships:
- [ ] Run copy through `/content-gate` (Four U's ≥ 10/16 for ads)
- [ ] Check platform TOS via `/ad-copy` policy reference
- [ ] A/B variant created (never launch a single version)
- [ ] Tracking UTMs configured
- [ ] Landing page matches creative promise
- [ ] Mobile preview approved by a human
