# Press Release Checklist

> **Use when:** Final review before distributing any press release.

## Structure
- [ ] "FOR IMMEDIATE RELEASE" header present
- [ ] Headline under 100 characters with action verb
- [ ] Subheadline adds key detail (optional)
- [ ] Dateline correct: CITY, State (Month Day, Year) —
- [ ] ### end mark present

## Lead Paragraph (5 Ws in 25-40 words)
- [ ] WHO: Company name included
- [ ] WHAT: Announcement clearly stated
- [ ] WHEN: Date in dateline
- [ ] WHERE: City in dateline (+ platform if relevant)
- [ ] WHY: Benefit/purpose stated
- [ ] SO WHAT: Reader relevance clear

## Quotes
- [ ] Executive quote included (2-3 sentences)
- [ ] Quote adds insight (doesn't repeat body text)
- [ ] Full name and title attributed
- [ ] No "I" at start of quote
- [ ] No buzzwords (synergy, leverage, optimize)

## AP Style
- [ ] Numbers: Spell out 1-9, numerals 10+
- [ ] Dates: Abbreviated months (Jan., Feb., Sept.)
- [ ] No Oxford comma
- [ ] Titles before name capitalized, after lowercase
- [ ] Acronyms defined on first use

## Crypto/Fintech (if applicable)
- [ ] Metrics verifiable (TPS, TVL, user count)
- [ ] Integration partners named
- [ ] Self-custody emphasized
- [ ] No hype words (revolutionary, groundbreaking)
- [ ] No price predictions or investment advice
- [ ] Trust signals included (audits, investors)

## Boilerplate & Contact
- [ ] About [Company] section (3-4 sentences)
- [ ] Media contact complete (Name, Email, Phone)
- [ ] Website URL included
- [ ] Social handles listed

## Distribution
- [ ] All facts and figures verified
- [ ] Links tested
- [ ] High-res images available (300 DPI+)
- [ ] Executive headshots prepared
- [ ] Timing confirmed (Tue-Thu, 9-11am optimal)
- [ ] Target publication list ready
