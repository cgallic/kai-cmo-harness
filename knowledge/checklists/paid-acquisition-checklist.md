# Paid Acquisition Checklist

> Complete before launching or optimizing any paid acquisition campaign.

---

## Pre-Launch Checklist

### Unit Economics Validation
- [ ] Calculate target CAC based on LTV (target LTV:CAC > 3:1)
- [ ] Define CAC Payback period target (< 12 months ideal)
- [ ] For eCommerce: Calculate CM1, CM2, CM3 for each product category
- [ ] Set variable ROAS targets by product margin profile
- [ ] For SaaS: Calculate Magic Number and New CAC Ratio

### Audience Architecture
- [ ] Define audience signals (customer lists, high-value purchasers, newsletter subscribers)
- [ ] Set up brand exclusion lists for PMax campaigns
- [ ] Configure suppression audiences (exclude current customers from acquisition)
- [ ] For B2B: Define ABM tiers (Tier 1/2/3) and account lists
- [ ] Set up intent data layering (if available)

### Creative Preparation
- [ ] Develop at least 3 hook variations using the 7-Hook Framework
- [ ] Create 3:2:2 test structure (3 creatives, 2 texts, 2 headlines)
- [ ] Apply PAS framework to all primary text copy
- [ ] Headlines under 40 characters
- [ ] Specific CTAs (not generic "Submit" or "Click Here")
- [ ] For TikTok: Ensure native, non-ad feel ("Make TikToks, not ads")

### Landing Page Alignment
- [ ] Dedicated landing pages for each major ad campaign
- [ ] Message match between ad hook and landing page headline
- [ ] Clear, singular CTA on landing page
- [ ] Mobile optimization verified
- [ ] Page load time < 3 seconds

### Tracking & Measurement Setup
- [ ] Conversion API (CAPI) configured for Meta
- [ ] Enhanced Conversions enabled for Google
- [ ] UTM parameters standardized across campaigns
- [ ] Attribution window settings documented
- [ ] Baseline metrics captured for incrementality comparison

---

## Platform-Specific Checklist

### Google Ads
- [ ] Account structure follows Hagakure/STAG method (consolidated)
- [ ] Minimum 30-50 conversions/month per campaign for Smart Bidding
- [ ] Negative keyword lists applied
- [ ] Brand Exclusion list in PMax campaigns
- [ ] Asset Groups organized by product category/landing page
- [ ] New Customer Acquisition (NCA) goal enabled (if available)

### Meta (Facebook/Instagram)
- [ ] ASC campaign for scaling proven creative
- [ ] Separate manual campaigns for testing
- [ ] Customer Budget Cap set (< 10% spend on existing customers)
- [ ] Advantage+ Creative enabled for asset optimization
- [ ] Remix variations planned for winning creative (5x hook variations)

### TikTok
- [ ] Spark Ads enabled for organic post boosting
- [ ] Search Ads Toggle activated
- [ ] TikTok Shop integration (if applicable)
- [ ] Creator/UGC content sourced

### LinkedIn
- [ ] Thought Leader Ads from employee profiles (not just company page)
- [ ] Document Ads for MOFU content
- [ ] Matched Audiences configured (website retargeting, CRM lists)
- [ ] Lead Gen Forms tested against landing page forms

### Programmatic/DSP
- [ ] Supply Path Optimization (SPO) verified
- [ ] IVT/fraud detection enabled
- [ ] Brand safety controls configured
- [ ] CTV: VCR and CPCV as primary metrics (not CTR)

---

## Ongoing Optimization Checklist

### Weekly
- [ ] Search term audit (Google) - add negatives for irrelevant queries
- [ ] Pacing check - budget utilization on track
- [ ] CPA/ROAS trend review by campaign
- [ ] Creative fatigue check (frequency > 10 = refresh needed)
- [ ] Bid adjustments based on performance

### Bi-Weekly
- [ ] Creative refresh - launch new hook variations
- [ ] Audience performance review - pause underperformers
- [ ] Landing page conversion rate check
- [ ] Competitor ad library review

### Monthly
- [ ] Full funnel analysis - impressions to revenue
- [ ] Channel efficiency comparison
- [ ] Budget reallocation based on incremental performance
- [ ] New creative production pipeline review

### Quarterly
- [ ] MMM analysis for macro budget allocation
- [ ] Geo-lift or incrementality test (if budget allows)
- [ ] LTV:CAC recalculation with updated data
- [ ] CAC Payback recalculation
- [ ] Strategic channel mix review

---

## Measurement & Reporting Checklist

### Attribution Hygiene
- [ ] Platform attribution compared against internal analytics
- [ ] Triangulation: MTA + MMM + Incrementality cross-referenced
- [ ] Branded vs. Non-branded performance separated
- [ ] New customer vs. returning customer metrics isolated

### Incrementality Testing
- [ ] Test and Control regions identified (for geo-lift)
- [ ] Baseline conversion data collected (4+ weeks)
- [ ] Statistical significance calculator prepared
- [ ] Results documented and used to calibrate MMM

### Reporting Standards
- [ ] Incremental CAC (iCAC) reported alongside blended CAC
- [ ] Contribution Margin reported (not just ROAS)
- [ ] CAC Payback Period tracked monthly
- [ ] Platform-reported vs. verified metrics clearly labeled

---

## Anti-Pattern Prevention Checklist

- [ ] Branded search segmented from non-branded (avoid ROAS Illusion)
- [ ] Retargeting spend capped (< 15-20% of total budget)
- [ ] No SKAG structures (using STAG/Hagakure instead)
- [ ] Creative refresh schedule defined (every 2-4 weeks)
- [ ] Post-click experience matches ad promise
- [ ] Algorithm has sufficient data (30-50 conversions/month minimum)
- [ ] No "set and forget" - active management scheduled

---

## Launch Approval

- [ ] Unit economics validated
- [ ] Tracking confirmed working
- [ ] Creative approved and loaded
- [ ] Landing pages live and tested
- [ ] Budget and bid strategy confirmed
- [ ] Measurement baseline captured
- [ ] Go/No-Go decision documented
