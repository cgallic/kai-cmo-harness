# 2026 Marketing Readiness Checklist

> **Use when:** Preparing for FY2026 strategic planning, auditing marketing infrastructure for AI/privacy changes, or assessing organizational readiness for the Agentic Economy.

---

## Data Infrastructure

### First-Party Data Strategy
- [ ] Customer Data Platform (CDP) implemented and collecting consented data
- [ ] Data Clean Room solution selected (OPJA-compliant for interoperability)
- [ ] Zero-party data collection mechanisms in place (quizzes, preference centers)
- [ ] Privacy-value exchange articulated ("Share X, receive Y")
- [ ] UID2 or equivalent deterministic identifier strategy defined

### Data Governance
- [ ] Data warehouse centralized (Snowflake/BigQuery)
- [ ] Reverse ETL implemented (tools read from warehouse)
- [ ] Single source of truth established for all AI agents
- [ ] Data structuredness audit completed (machine-readable formats)

---

## AI & Search Discoverability

### Knowledge Graph Presence
- [ ] Brand entity verified in Wikidata
- [ ] Company profile accurate in Crunchbase
- [ ] Structured data (Schema.org) implemented across website
- [ ] FAQ Schema applied to all content pages

### Generative Engine Optimization (GEO)
- [ ] AI Overview visibility audited (tools: Peec AI, Profound)
- [ ] "Share of Model" baseline established for category queries
- [ ] Citation-worthy content strategy defined (fact-dense, logically coherent)
- [ ] Long-tail question content created for RAG triggering

### Content Authentication
- [ ] C2PA Content Credentials implementation planned
- [ ] Human-created content labeling strategy defined
- [ ] "Human Premium" content identified and protected

---

## Media & Measurement

### Attention Metrics
- [ ] Attention measurement vendor selected (Adelaide, DoubleVerify, Playground xyz)
- [ ] Baseline attention scores established for current media
- [ ] Agency contracts updated to include aCPM KPIs
- [ ] Low-quality programmatic inventory audited/reduced

### Attribution Evolution
- [ ] Multi-touch attribution model implemented
- [ ] Incrementality testing framework established
- [ ] Last-click attribution dependencies removed
- [ ] Clean Room measurement capabilities activated

### New Metrics Tracking
- [ ] AI Citation Score tracking implemented
- [ ] Attention CPM (aCPM) reporting enabled
- [ ] Share of Model (SoM) monitoring in place
- [ ] Open rate metrics de-prioritized (AI filter impact)

---

## Creative & Content Operations

### Creative Velocity
- [ ] Modular creative production tools implemented (Celtra, Smartly)
- [ ] Creative testing framework documented (Pre-flight → BAU Challenge → Scale)
- [ ] 2-4 new variants per cycle capacity established
- [ ] Brand guideline guardrails defined for AI agents

### Content Strategy
- [ ] Video-first production workflow implemented
- [ ] AI transcription/repurposing pipeline built
- [ ] Founder/leadership direct channel launched
- [ ] Synthetic vs authentic content strategy defined

### Agentic Workflows
- [ ] Multi-agent workflow piloted (LangChain, Zapier AI)
- [ ] Agent governance framework established (brand safety, ethics)
- [ ] "AI Core" team or function defined
- [ ] AI orchestration skills assessed/developed

### Paid Media

- [ ] AI content labeling requirements monitored

## Scenario Preparedness

### If AI Regulation Tightens
- [ ] C2PA watermarking ready to deploy
- [ ] "Human-Made" premium content identified

### If Economic Downturn
- [ ] Retention budget reallocation plan ready
- [ ] LTV-focused messaging prepared
- [ ] Empathy messaging templates created

### If Platform Fails (X, TikTok)
- [ ] Life raft strategy documented
- [ ] Owned channel migration plan ready
- [ ] Alternative platform presence established (Pinterest, Snap, Shorts)

### If Privacy Regulation Expands
- [ ] Data Clean Room strategy activated
- [ ] Zero-party data collection scaled
