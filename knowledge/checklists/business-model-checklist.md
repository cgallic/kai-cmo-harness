# Business Model Marketing Checklist

> **Use when:** Setting up marketing strategy for a specific business model, auditing current approach against benchmarks, or validating channel/tactic priorities.

---

## Universal Pre-Work (All Models)

- [ ] Unit economics defined (CAC target, LTV goal, payback period)
- [ ] Primary constraint identified (Awareness? Trust? Pipeline? Liquidity?)
- [ ] Customer journey mapped (Where do they learn? Decide? Buy?)
- [ ] North Star metric selected
- [ ] Team structure matched to stage (Generalist vs Specialist)
- [ ] Budget allocation benchmarked against model norms

---

## B2B SaaS Checklist

### Strategy Foundation
- [ ] Pipeline velocity defined as primary metric
- [ ] Marketing-sourced pipeline target set (30-60% of revenue)
- [ ] MQL/SQL/SAL definitions documented and agreed with Sales
- [ ] SLA with Sales includes response time (<5 min ideal)
- [ ] Feedback loop established (rejection reason codes)
- [ ] PLG vs Sales-Led motion determined

### Funnel Health
- [ ] Visitor-to-Lead rate tracked (target: 2-5%)
- [ ] Lead-to-MQL rate tracked (target: 31-41%)
- [ ] MQL-to-SQL rate tracked (target: 15-21%, best: 40%)
- [ ] SQL-to-Opportunity rate tracked (target: 36-42%)
- [ ] Opportunity-to-Close rate tracked (target: 20-30%)

### Channel Execution
- [ ] LinkedIn Ads for decision-maker targeting
- [ ] Google Ads for high-intent capture
- [ ] SEO prioritizing purchase intent keywords
- [ ] ABM program implemented (1:1, 1:Few, or 1:Many)
- [ ] Intent data integrated for prioritization
- [ ] Review sites presence (G2, Capterra)

### Content Strategy
- [ ] TOFU content: "What is" / "How to"
- [ ] MOFU content: Buying guides, comparisons, webinars
- [ ] BOFU content: ROI calculators, case studies, implementation guides
- [ ] Gating strategy defined (high-value utility only)
- [ ] Case studies follow Problem → Solution → Quantitative Results

### Anti-Patterns Avoided
- [ ] Not gating all content
- [ ] Not passing raw MQLs to AEs (SDR qualification in place)
- [ ] Tracking Pipeline Revenue, not just "Leads"

## DTC E-Commerce Checklist

### Strategy Foundation
- [ ] Brand + Margin + Data equation understood
- [ ] Gross margin validated (>50-60% for paid acquisition math)
- [ ] 60-day LTV calculated and tracked
- [ ] LTV:CAC ratio monitored (target: >3:1)
- [ ] Omnichannel expansion planned (wholesale, marketplace)

### Acquisition Channels
- [ ] Meta Advantage+ Shopping campaigns running
- [ ] Google Shopping and branded search defended
- [ ] TikTok native content strategy (lo-fi, trend-driven)
- [ ] TikTok Shop operational (if applicable)

### Creative Operations
- [ ] Creative testing framework implemented
- [ ] 2-4 new variants tested per cycle
- [ ] Thumb Stop Ratio tracked
- [ ] Hold Rate tracked
- [ ] UGC sourcing systematized
- [ ] Whitelisting/licensing for creator ads

### Retention Engine
- [ ] Email revenue target: 20-30% of total
- [ ] SMS for time-sensitive (drops, flash sales)
- [ ] Email for storytelling and education
- [ ] Subscribe & Save for consumables
- [ ] Loyalty program (experiential, not just points)
- [ ] Win-back flows: 60, 90, 120 days

### Benchmarks Tracked
- [ ] Blended CAC monitored
- [ ] Conversion rate by category benchmarked
- [ ] Repeat Purchase Rate tracked (target: 25-30%)
- [ ] Meta iROAS tracked (median: ~2.31x)

## Fintech Checklist

### Strategy Foundation
- [ ] Trust deficit acknowledged and addressed
- [ ] Compliance budget allocated (8-20% of revenue)
- [ ] Pre-approved claims library created
- [ ] Legal review process streamlined
- [ ] Education-first content strategy defined

### Trust Building
- [ ] Social proof prominent (reviews, testimonials)
- [ ] Security badges displayed
- [ ] Insurance/protection highlighted
- [ ] Transparent pricing/terms
- [ ] "Money 101" educational content

### Channel Execution
- [ ] Referral program implemented (target CAC: $5-15 viral)
- [ ] Content marketing for high-volume financial queries
- [ ] Partnership/embedded finance opportunities evaluated
- [ ] DevRel for B2B infrastructure products

### Compliance
- [ ] UDAAP compliance verified (no "Guarantee," "Best" without proof)
- [ ] ECOA compliance: targeting not discriminatory
- [ ] Disclosures "clear and conspicuous" on all ads
- [ ] Copy pre-screening for non-compliant terms

### Vertical-Specific
- [ ] Consumer: Viral loops, referrals, brand affinity
- [ ] B2B: ABM, content, DevRel
- [ ] Lending: Fair lending, transparent disclosure

### Benchmarks Tracked
- [ ] CAC by segment (Consumer viral: $5-15; B2B: ~$1,450)
- [ ] Retention rate monitored (industry low: ~37%)
- [ ] App abandonment tracked (benchmark: 73% first week)
- [ ] Activation/funding rate as North Star

## Crypto/Web3 Checklist

### Strategy Foundation
- [ ] Community-first philosophy embedded
- [ ] Token incentive strategy defined (airdrops, points)
- [ ] Bear vs Bull market playbook selected
- [ ] Decentralization vs execution balance determined

### Community Building
- [ ] Twitter/X presence for announcements/thought leadership
- [ ] Discord structured (roles, channels, governance)
- [ ] Telegram for international/announcements
- [ ] Ambassador program incentivized
- [ ] Bot prevalence monitored in community metrics

### Token Strategy
- [ ] Airdrop strategy: Retention-based vesting (not dump-prone)
- [ ] Points program for pre-token engagement
- [ ] Token distribution health monitored (Gini coefficient)

### Developer Relations (Infrastructure)
- [ ] Hackathon participation/sponsorship
- [ ] Grants program established
- [ ] Developer documentation quality

### PR & Legitimacy
- [ ] Crypto media coverage (Decrypt, CoinDesk, The Block)
- [ ] Thought leadership content

### Regulatory
- [ ] Howey Test implications reviewed
- [ ] No promises of "profits" from "efforts of others"
- [ ] Geo-fencing for US (if needed)
- [ ] Legal review of all token-related marketing

### Benchmarks Tracked
- [ ] Active wallet growth (not just signups)
- [ ] TVL growth (DeFi)
- [ ] Active chatting users (not total members)
- [ ] CAC per wallet (target: <$50)

## Marketplace Checklist

### Strategy Foundation
- [ ] Cold start strategy defined
- [ ] Supply vs demand priority determined
- [ ] Geographic/category constraint applied
- [ ] Subsidization strategy for valuable side
- [ ] Single-player mode evaluated (SaaS tool first)

### Liquidity Focus
- [ ] Buyer liquidity tracked (search-to-fill rate)
- [ ] Seller liquidity tracked (utilization rate)
- [ ] Match Rate monitored (target: >60%)
- [ ] Churn correlation to liquidity understood

### Growth Tactics
- [ ] Hardest side acquired first
- [ ] Launch market constrained (one city/category)
- [ ] Manual supply "faking" if needed early stage
- [ ] Managed → Open transition planned

### Unit Economics
- [ ] Take rate competitive for category
- [ ] Buyer CAC tracked (~$30 benchmark)
- [ ] Seller CAC tracked (~$150 benchmark)
- [ ] LTV:CAC >3:1

### Benchmarks Tracked
- [ ] Match Rate (>60% healthy)
- [ ] GMV growth (15-25% YoY)
- [ ] Take rate vs category benchmarks

## Team Structure by Stage

### Seed (0-20 employees)
- [ ] 1 Full Stack Marketer or Founder-led
- [ ] Focus: Experimentation, PMF validation
- [ ] Structure: Flat, reports to CEO

### Series A (~50 employees)
- [ ] Director + Demand Gen + Product Marketing + Content
- [ ] Focus: Repeatable engine
- [ ] Structure: Functional specialization

### Series B (~125 employees)
- [ ] VP + Directors (Brand, Demand, Product) + Specialists
- [ ] Focus: Specialization and scale
- [ ] Structure: Dedicated creative, ops, analytics

## Budget Benchmarks

| Model | Budget as % Revenue |
|-------|---------------------|
| B2B SaaS (Growth) | 10-20% |
| B2B SaaS (VC Hyper-growth) | Up to 47% |
| DTC E-commerce | 20-30% |
| Fintech | Includes 8-12% compliance |

---

## Universal Anti-Patterns

- [ ] Not tracking unit economics
- [ ] Optimizing for vanity metrics
- [ ] Mismatched team structure for stage
- [ ] No clear North Star metric
- [ ] Channel strategy not matched to model
- [ ] Ignoring retention for acquisition
