# Social Media Audit Checklist

> **Use when:** Evaluating an existing social media presence, onboarding a new client, or planning a strategy refresh.

---

## Profile Audit (Per Platform)

### Branding Consistency
- [ ] Profile photo: high-quality, recognizable at small sizes (logo or headshot)
- [ ] Cover/banner image: current, matches brand, correct dimensions
- [ ] Bio/description: clear value prop, keywords, CTA
- [ ] Link in bio: working, points to correct destination (landing page > homepage)
- [ ] Contact information: email, phone, address (where applicable)
- [ ] Username: consistent across platforms (or documented if different)
- [ ] Verified/premium status: applied for where available

### Content Audit
- [ ] Last post date: when was the last post? (>30 days = inactive)
- [ ] Posting frequency: how often? (compare to platform benchmarks)
- [ ] Content mix: what % is promotional vs educational vs entertaining?
- [ ] Content quality: passes visual/copy quality standards?
- [ ] Engagement rate: above or below platform benchmark?
- [ ] Response rate: how quickly are comments/DMs answered?
- [ ] Top 5 posts (last 90 days): what type of content performs best?
- [ ] Bottom 5 posts (last 90 days): what underperforms?

### Audience Analysis
- [ ] Follower count: growing, flat, or declining?
- [ ] Follower demographics: match target persona?
- [ ] Fake follower check: engagement rate vs follower count (low ratio = bot followers)
- [ ] Audience overlap: how much overlap between platforms?

## Platform-Specific Checks

### Instagram
- [ ] Reels: posting regularly? (algorithm priority)
- [ ] Stories: daily stories? (engagement with existing followers)
- [ ] Highlights: organized, current, useful?
- [ ] Shopping: enabled (if applicable)?
- [ ] Hashtag strategy: documented? Working?
- [ ] Grid aesthetic: intentional or random?

### LinkedIn
- [ ] Company page: fully complete (all sections filled)?
- [ ] Employee advocacy: are team members sharing company content?
- [ ] Article publishing: using LinkedIn's long-form publishing?
- [ ] Showcase pages: used for sub-brands (if applicable)?
- [ ] Posting from personal profiles vs company page?

### X/Twitter
- [ ] Thread strategy: creating threads for reach?
- [ ] Reply engagement: responding to mentions and replies?
- [ ] List management: curated lists for monitoring?
- [ ] Spaces: hosting or participating in live audio?
- [ ] Pinned tweet: current, high-performing, with CTA?

### TikTok
- [ ] Video frequency: 1-3x/day?
- [ ] Trending sounds: using platform trends?
- [ ] Duets/stitches: engaging with other creators?
- [ ] Profile link: points to landing page?
- [ ] TikTok Shop: connected (if applicable)?

### YouTube
- [ ] Channel art: professional, current?
- [ ] Channel description: keyword-optimized?
- [ ] Playlists: organized by topic?
- [ ] End screens and cards: on every video?
- [ ] Community tab: used for polls and updates?
- [ ] Shorts: publishing alongside long-form?

### Facebook
- [ ] Page info: hours, location, category correct?
- [ ] Reviews: responding to reviews?
- [ ] Events: using for launches/webinars?
- [ ] Group: running a Facebook Group (if applicable)?
- [ ] Messenger: auto-replies configured?

### Content Gaps
